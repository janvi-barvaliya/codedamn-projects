# 🚀 HoverCounter

## A simple React app that counts how many times a user hovers over a button 🖱️

## ✨ Features

- 🎯 Hover over the button to increase the count.
  
- ⚛️ Built using React and hooks.
  
- 🎨 Clean and minimal styling with CSS.
